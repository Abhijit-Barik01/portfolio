### Option 1: GitHub Pages (Recommended)

1. **Create a new repository** on GitHub named `abhijit-barik.github.io` or any name you prefer
2. **Push the code**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Portfolio website"
   git branch -M main
   git remote add origin https://github.com/Abhijit-Barik01/abhijit-barik.github.io.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to repository Settings â†’ Pages
   - Select `main` branch as source
   - Click Save
4. **Access your site**: `https://abhijit-barik01.github.io/`

### Option 2: Vercel

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```
2. Deploy:
   ```bash
   vercel
   ```
3. Follow the prompts and your site will be live!

### Option 3: Netlify

1. Drag and drop the folder to [Netlify Drop](https://app.netlify.com/drop)
2. Your site is live instantly!
